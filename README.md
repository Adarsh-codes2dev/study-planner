# StudyPlanner - AI-Powered Study Schedule Generator

An AI-powered app that analyzes your course syllabus and generates a personalized, day-by-day study schedule.

## Features
- Syllabus Analysis - extracts all assignments, exams, deadlines and topics automatically
- Smart Scheduling - generates a realistic week-by-week study plan
- Deadline Tracker - color-coded overview of all assignments and exams
- Study Tips - personalized tips based on your course content
- Cross-Platform - Android mobile app and macOS desktop app

## Platforms
- Android: Flutter + Dart
- macOS Desktop: Electron + React + Vite

## AI Powered By
Google Gemini 2.5 Flash API

## Setup
Add your Gemini API key from https://aistudio.google.com

### Android
```bash
cd study_planner
flutter pub get
flutter run
```

### macOS Desktop
```bash
cd study-planner
npm install
npm run dev
```

## Built With
- Flutter, Electron, React, Vite, Google Gemini API
